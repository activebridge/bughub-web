Getting started

First of all create Create a project

Invite collaborators

  1. Navigate to project's access tab
  2. Enter your collaborator email to invite or add by github email if he is signed up on Bugno.io already
  3. Invited collaborator should sign up with registration token

Subscribe a plan

  1. Navigate to project's subscription tab
  2. Choose most adorable plan
  3. Enter your credit card credentials

And of course install sdk for rails or javascript
