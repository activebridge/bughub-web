# Getting started ## How to create project 1. Navigate to dashboard, click "New" on the sidebar. 2. Enter your project name and description if you wish. ## How to invite collaborators 1. To invite your collaborators navigate to access tab. 2. Enter your collaborator by email if he/her is not signed up yet or his/her github associated email. 3. Invited collaborator should sign in with registration token ## Manage subscription 1. To start capturing exceptions to have to add a subscription. 2. Navigate to subscription tab. 3. Choose most adorable plan. 4. Enter your credit card credentials.
